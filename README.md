# Estudio Personal — Masterclasses IA Aplicada

Sistema de estudio estructurado en 4 masterclasses sobre IA aplicada al dominio
regulatorio y fiscal chileno. Combina teoría, código ejecutable y ejemplos prácticos
con datos de normativa, presupuesto público y Diario Oficial.

## Arranque rápido

```bash
git clone <repo-url> && cd estudio-personal
uv sync
cp .env.example .env   # completar con API keys reales
```
